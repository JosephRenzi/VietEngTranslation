### VietEngTranslation
This project is a robust, context-aware multi-agent translation system that orchestrates a self-correcting Reflection Loop between specialized Translator and Critic agents, supported by dynamic context inference, RAG-based glossary tools, and comprehensive SQLite logging for observability.

# How to run this
1. Ensure you have your Google_API_KEY available.
2. Run pip install -r requirements.txt
3. Execute python main.py
